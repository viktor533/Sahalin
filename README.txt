В настоящий момент недописанный проект. 

Предполагалось создание универсальной площадки, для создания игр, подобный стратегии "Сахалин". 
С возможностью редактировать типы местности, ресурсов, стоимости зданий, их износа и т.д. 
Идея заключалась в создании максимально гибкой и удобной платформы, для всех игр вида "строить и выжить".
К сожалению, проект зачах в стадии разработки.
